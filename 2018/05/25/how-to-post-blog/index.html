how to post blog

撰写于 2018-05-25 修改于 2018-05-25

**


 1.在github中新建一个仓库 以用户名.github.io 命名
 2.修改_config.yml 81-83
             type: git
              repository: git@github.com:myuer/myuer.github.io.git
              brach: master
 3.在hexo文件夹中，右键打开git bash，输入hexo g
         npm install hexo-deployer-git –save
   3.1 403 报错说明将repository：将http改成 ssh的地址。
         remote: Permission to myuer/myuer.github.io.git denied to NicolasGui.
         fatal: unable to access ‘https://github.com/myuer/myuer.github.io.git/': The requested URL returned error: 403

3.2 报错：
$ hexo d
ERROR Deployer not found: git
之后输入 npm install hexo-deployer-git –save

3.3 之后hexo g
hexo d
出现INFO Deploy done: git
说明blog被曝到 网上了。
